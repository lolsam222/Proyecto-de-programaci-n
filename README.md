# Buscaminas

## Integrantes
- Samuel Venegas Bernal - svenegasb@unal.edu.co

## Descripción
Juego de buscaminas clasico, ejecutable en la terminal
compatible para windows y linux con 3 dificultades y con guardado
y carga de partida.

## Compilación y ejecución
para windows:
ejecutarlo en DEV-C++

en la terminal misma:
buscaminas.exe

debes estar en la misma carpeta donde esta el .exe.

tambien para :

.\buscaminas.exe

para linux:
g++ -o buscaminas main.cpp

si no sirve el primer comando:
./buscaminas
